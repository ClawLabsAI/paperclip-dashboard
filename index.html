Instalación automática
Un solo comando te lleva de cero a plataforma de agentes IA en funcionamiento.
Opción 1 — Onboard automático (recomendado)
$ npx paperclipai onboard --yes 📋 Copiar
Bootstraps toda la instalación automáticamente. API en localhost:3100
Opción 2 — Manual
clawlabsai — setup
$ git clone https://github.com/paperclipai/paperclip $ cd paperclip $ pnpm install $ pnpm dev # API disponible en http://localhost:3100 ✓ Paperclip running · PostgreSQL embedded · ready $
⚠ Para uso diario usa pnpm paperclipai run — más rápido e incluye auto-healing.

🚀 Quickstart
1. Instala OpenClaw
npm install -g @openclaw/cli
2. Inicia el servidor
openclaw serve --port 3100
3. Conecta ClawLabsAI
Introduce tu endpoint y API key en la pestaña Configuración y pulsa "Guardar y conectar"
🔌 API Reference
GET/agentsLista agentes
POST/agents/:id/runEjecutar agente
GET/heartbeatEstado sistema
POST/tasksCrear tarea
GET/metricsMétricas uso
WS/streamLive stream
