Paso 1 / 5
Instalar Paperclip
Un solo comando instala y configura todo: servidor API, base de datos embebida y primera empresa.
Opción A — Quickstart (recomendado)
npx paperclipai onboard --yes
↳ Instala deps · configura DB embebida · inicia servidor en http://localhost:3100
Opción B — Instalación manual
git clone https://github.com/paperclipai/paperclip.git
cd paperclip && pnpm install
pnpm dev
bash — paperclip installation
